# Git 快速保存 Skill

请执行以下 Git 操作：

1. **检查状态**：运行 `git status` 查看当前更改
2. **添加更改**：使用 `git add .` 添加所有更改
3. **创建提交**：使用 `git commit -m "update: $(date)"` 或 `git commit -m "chore: 快速保存"` 创建提交
4. **推送到远程**：运行 `git push` 推送到远程仓库
5. **确认结果**：显示操作结果

## 使用场景
- 快速保存当前工作进度
- 不需要详细提交信息的临时保存
- 自动保存工作状态

## 使用示例
```
/save
```
